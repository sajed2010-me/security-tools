# security-tools
A collection of Python security reconnaissance tools.
This first tool provides headers that might be of use for pentesters, i will hopefully add features other than its now basic structure to make it better.
## Changelog

- **1st update (1st-tool.py – 31/03/2026)**  
  Added multiple improvements:
  - Now checks for faulty URLs that may contradict the behavior of the `requests` library.
  - Handles timeouts and connection errors when a request takes too long or fails.
  - Added severity classes that can be useful for security auditors to prioritize findings.

  Thank you very much for taking the time to check out this very humble project.
